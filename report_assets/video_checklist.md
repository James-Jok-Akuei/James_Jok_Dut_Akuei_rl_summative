# Video Demonstration Checklist

- Turn your camera on before you start recording.
- Share your entire screen, not a single window.
- Start by stating the problem in one or two sentences.
- Explain what the drone agent represents in the mission.
- Explain the reward structure briefly.
- State the objective clearly: pickup, navigate safely, manage battery, and deliver on time.
- Run the best-performing model using `python3 main.py`.
- Keep the terminal visible so verbose outputs can be seen.
- Keep the Pygame GUI visible at the same time.
- End with a short interpretation of the agent's performance and any remaining limitations.
